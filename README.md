### Sign Language Detector
This project uses live webcam footage to detect sign language hand gestures made by the user, then identifies and displays the corresponding letter in real time.

## Built With
- Python3 3.9.6
- OpenCV 4.12.0
- MediaPipe 0.10.14
- NumPy 2.0.2
- scikit-learn 1.6.1
